# NetSuite Leave Management
Leave Management Module customized for NetSuite System

**Technologies:**
- Netsuite, SuiteScript 2
- TypeScript
- Jest

## Getting Started
Using TypeScript for SuiteScript 2.0 

**Transpile Project**

```
npm transpile
```
_OR_
```
tsc
```

### Testing
**Run tests**
```
npm test
```
